Election Management Tips

Quick Guide

To set up a complete election:

  1. Create a new election using the form
  2. Add candidates to the election from the election list
  3. Set the election as active when ready for voting
  4. View results after the election end time
Important Notes
  • Elections without candidates cannot be voted on
  • Set realistic start and end times
  • Make sure to add all candidates before the election starts
  • Results are visible to everyone after the end time
